# Dalbovač (Next.js + Tailwind)

Kompletní funkční skeleton inspirovaný stylem Klik.cz.

## Rychlý start
```bash
npm i
npm run dev
```

## Build
```bash
npm run build
npm start
```

> Pokud nasazuješ na Vercel a měl jsi dřív chybu `Couldn't find any pages or app directory`, tak tady je `app/` složka připravena.
